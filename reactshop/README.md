安装与运行程序
-------------
克隆项目
#### `> $ git clone git@github.com:yiran8080/git_demo.git`

在当前目录下安装依赖包
#### `> $ npm install`

在开发者模式下启动Webpack服务
#### `> $ npm run start`

在生产环境下启动Webpack服务
#### `> $ npm run build`

在开发者模式运行后，打开浏览器 http://localhost:3000/#/home/index 运行项目。
