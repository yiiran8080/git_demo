React-Demo
====
基于Reactjs && ES6 && Webpack构建移动端电商平台页面交互
-----
* 首页

* 商品详情页
* 购物车页面
* 商品结算页
* 生成订单号
* 订单管理页

克隆项目
#### `> $ git clone `

在当前目录下安装依赖包
#### `> $ npm install`

在开发者模式下启动Webpack服务
#### `> $ npm run start`

在生产环境下启动Webpack服务
#### `> $ npm run build`

在开发者模式运行后，打开浏览器 http://localhost:3000/#/home/index 运行项目。
